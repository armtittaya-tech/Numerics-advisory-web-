01 ขอบเขตงาน

3 บริการสำหรับธุรกิจที่กำลังจะ scale

ไม่ใช่ทุกธุรกิจที่ต้องการบริการนี้ — แต่ถ้าถึงจุดที่ใช่ มันคือสิ่งที่กำหนดว่าคุณจะปิด deal ได้ในราคาที่ควร หรือเสียโอกาส

SA1

Business Valuation

ประเมินมูลค่าธุรกิจของคุณด้วยวิธีที่ investor / นายธนาคารยอมรับ — DCF, Comparable, Asset-based

  • 3 valuation methods พร้อมเหตุผล
  • Sensitivity analysis ของ assumption
  • Valuation memo 12–20 หน้า
  • Defense package สำหรับ Q&A กับ investor
  • เหมาะกับ: fundraising · M&A · partner buyout · estate planning
SA2

Growth Scenario Modeling

สร้าง financial model 3-statement ที่ลากไปได้ 3-5 ปี เพื่อตอบคำถาม "ถ้าทำ X จะเป็นไง"

  • 3-statement integrated model (P&L · BS · CF)
  • Base / Best / Worst case scenarios
  • Unit economics modeling
  • Capital structure analysis
  • เหมาะกับ: ขยายสาขา · ออกผลิตภัณฑ์ใหม่ · เข้าตลาดใหม่
SA3

Investment & Exit Readiness

เตรียมธุรกิจให้พร้อมรับ investor หรือผู้ซื้อ — diligence-ready ตั้งแต่วันแรก

  • Financial due diligence package
  • Data room organization
  • Quality of Earnings (QoE) analysis
  • Pitch deck financial section
  • เหมาะกับ: pre-Series A/B · trade sale · IPO ระยะแรก
02 ขั้นตอนการทำงาน

วิธีทำงานแบบ project-based

เริ่มได้ภายใน 1 สัปดาห์ ทำงานกับทีมเฉพาะของลูกค้า ไม่ใช่ส่งต่อให้พนักงานทั่วไป

สัปดาห์ 1Scope

ทำความเข้าใจ & วาง scope

คุย objective ของ project — fundraising ระดับไหน, ขาย exit ภายในกี่ปี, ขยายสาขาแบบไหน scope ที่ชัดคือครึ่งทางของงานที่ดี

สัปดาห์ 2–3Data

รวบรวมข้อมูล & ตรวจสอบ

ดึงข้อมูลย้อนหลัง 3 ปี + projection ของลูกค้า ตรวจสอบและทำความสะอาด ก่อนใช้ทำ model — ขยะเข้า ขยะออก

สัปดาห์ 4–8Build

สร้าง model · valuation · memo

ทำ financial model 3-statement, run valuation methods, เขียน memo ที่อ่านได้สำหรับ investor — ส่งร่างทุกสัปดาห์ ไม่ใช่ surprise ตอนสุดท้าย

สัปดาห์ 9–12Defense

ซ้อม Q&A + ปรับ assumption

ซ้อมตอบคำถามกับ investor / acquirer เตรียม defense package สำหรับ assumption ที่อาจถูกท้าทาย — เรานั่งฟัง pitch meeting ของคุณได้

03 สิ่งที่ลูกค้าได้รับ

Deliverables

01

Valuation Memo

เอกสาร 12-20 หน้า ที่ระบุ valuation 3 methods พร้อมเหตุผลและ assumption

02

Financial Model (Excel)

3-statement integrated model ที่ open ให้แก้ assumption ได้ — มี documentation

03

Scenario Analysis

Base / Best / Worst case พร้อม sensitivity ของ driver สำคัญ 5–8 ตัว

04

Pitch Deck Financials

5–8 หน้าของ pitch deck ในส่วน financial — number ตรงกับ model · พร้อมส่ง investor

05

Data Room Index

โครงสร้าง data room ตามมาตรฐาน VC/PE และ list เอกสารที่ต้องเตรียม

06

Q&A Defense Package

20–30 คำถามที่ investor มักจะถาม + คำตอบที่ตรงกับตัวเลขใน model

04 เหมาะกับใคร

เช็คความ fit ก่อนเริ่ม

เหมาะกับ เหมาะ

  • Founder ที่กำลังจะ raise pre-Series A / Series A
  • เจ้าของกิจการที่กำลังจะขาย หรือรับ strategic partner
  • SME ที่กำลังจะขยาย — เปิดสาขาใหม่ ลงทุนเครื่องจักร ลงทุน software
  • กิจการที่กำลังจะรับ investor เป็น minority ผู้ถือหุ้น
  • ครอบครัวที่กำลัง ส่งต่อกิจการ หรือ estate planning

อาจไม่ใช่ตอนนี้ ไม่ใช่

  • ธุรกิจที่ยังไม่มีงบ 12–24 เดือนของ track record — model จะเป็นเพียงการเดา
  • กิจการที่อยากได้ valuation สูงโดยไม่สนความสมจริง — เราจะปฏิเสธ
  • คนที่ต้องการแค่ "ตัวเลขสำหรับ pitch deck" — แต่ไม่ต้องการ rigor — เริ่มที่ Financial Intelligence แทน
05 คำถามที่พบบ่อย

FAQ

ราคาประมาณเท่าไหร่?

คิดเป็น project-based fee ขึ้นอยู่กับ scope (Valuation, Model, Readiness package — เลือกได้ทีละชิ้น) และ complexity ของธุรกิจ

หลังคุย 30 นาที จะส่ง scoping proposal ภายใน 5 วันทำการ พร้อม fixed fee ที่ระบุ deliverables ครบ

ต้องมีงบ audit แล้วถึงทำได้ไหม?

ไม่จำเป็นต้องมี audit แต่ งบที่ใช้ต้องสะอาด ถ้าไม่แน่ใจ เราเริ่มที่ บริการตรวจสอบบัญชี ก่อน เพื่อให้ valuation มีน้ำหนัก

Valuation ที่ทำให้ใช้ใน due diligence ของ investor ได้ไหม?

ใช้เป็นเอกสารเริ่มต้นของการคุยได้ — แต่ investor จะมี own DD process ของเขา (โดยเฉพาะ Series A ขึ้นไป) เรา position valuation ของเราเป็นจุดเริ่มของการเจรจา ไม่ใช่ final number

ทำได้พร้อมกันหลาย project ไหม?

ได้ — บาง engagement รวม Valuation + Modeling + Readiness ในแพ็กเกจเดียว ใช้เวลา 10–12 สัปดาห์ ถ้าเลือกแค่ Valuation อย่างเดียว 4–6 สัปดาห์

มี NDA / Confidentiality ไหม?

มีแน่นอน — เราเซ็น NDA ก่อนเริ่มงานทุกครั้ง ข้อมูลของลูกค้าไม่ใช้กับ engagement อื่น และเก็บแยกตาม project

หลัง project จบ ยังขอคำปรึกษาได้ไหม?

ได้ 3 เดือนแรกหลังส่งมอบฟรี — ตอบคำถามเรื่อง model หรือ valuation จาก investor หลังจากนั้น engage ใหม่หรือใช้ Financial Intelligence รายเดือนต่อ
